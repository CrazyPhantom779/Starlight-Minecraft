# Starlight-Minecraft

The official Minecraft server repository for **THE STARLIGHT NETWORK (SLN)**.

This repository contains the server configuration, modpack setup, and related resources used by the Starlight Minecraft server.

---

# Modpack Pull Request Requirements

Before submitting a mod or modpack-related pull request, ensure the following requirements are met:

## Required Version Information
- **Minecraft Version:** `1.21.1`
- **Mod Loader:** `NeoForge`

## Pull Request Guidelines
- Ensure all added mods are compatible with:
  - Minecraft `1.21.1`
  - `NeoForge`
- Do not include broken, experimental, or unstable builds unless explicitly approved by a maintainer/dev.
- Avoid duplicate functionality with existing mods.
- Try to avoid unoffical versions of modpacks.
- Provide a short description of:
  - What the mod does
  - Why it should be added
  - Any dependencies required
- If applicable, include:
  - Performance impact notes
  - Server/client-side requirements
  - Configuration changes
